# Smartphone Addiction Risk Predictor

A deep learning based mobile app that predicts smartphone addiction risk using LSTM.

## Tech Stack
- React Native (Expo)
- Flask (Backend)
- TensorFlow/Keras (LSTM)

## Features
- Predict addiction risk (Low / Medium / High)
- Real-time API integration
- Interactive mobile UI

## How to Run

### Backend
pip install -r requirements.txt
python app.py

### Frontend
cd addictionApp
npm install
npm start


---

## Architecture
Mobile App → Flask API → LSTM Model
